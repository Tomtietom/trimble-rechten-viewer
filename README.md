# Trimble Connect Rechten Viewer

Visueel overzicht van maprechten, groepen en gebruikers in Trimble Connect projecten.

## Gebruik

### Als Trimble Connect Extension
1. Ga naar je project in Trimble Connect → **Instellingen → Extensions**
2. Voeg de manifest-URL toe: `https://Tomtietom.github.io/trimble-rechten-viewer/manifest.json`
3. Activeer de extensie — de viewer verschijnt in de sidebar

### Standalone (zonder TC)
Open `index.html` in je browser en upload de CSV + Team Excel exports.

## Features
- Dashboard met samenvatting
- Rechtenweergave per groep, gebruiker en map
- Excel rechtenmatrix export
- Detectie van onbereikbare mappen (alleen via directe link)
- Uitleg hoe Trimble Connect rechten werken
